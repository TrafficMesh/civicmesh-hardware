# Node Assembly Guide

Estimated time: 30 minutes per unit

## Step 1: Prepare Carrier Board
Install Raspberry Pi CM4 into SO-DIMM slot

## Step 2: Install USB Hub
Mount and connect to RPi USB

## Step 3: Connect Peripherals
- Camera → CSI-2
- Modem → USB port 1
- OBD-II → UART0
- GPS → UART1

## Step 4: Power Management
Install buck converter and battery backup

## Step 5: Enclosure
Place board in weatherproof enclosure
